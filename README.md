# 🎂 Birthday Celebration Web App

An interactive, animated front-end web application created to celebrate a special birthday. This project combines smooth animations, custom styling, audio integration, and dynamic JavaScript effects to create an engaging digital greeting experience.

## 🚀 Live Demo
🔗 **[Experience the Live Web App](https://her-birthday-seven-azure.vercel.app/)**

---

## ✨ Features
*   **🎵 Background Music:** Integrated audio playback (`aud.mp3`) to set a celebratory mood.
*   **✨ Dynamic Effects & Animations:** Utilizes JavaScript logic (including specialized script files and jQuery) to handle fluid interactive states and visually unique transitions.
*   **🎨 Personalized Media Elements:** Features customized imagery (`IMG_1172.PNG`, `img.png`, `img2.png`) tailored specifically for the celebration layout.
*   **📱 Responsive Layout:** Structured using semantic HTML5 and custom CSS layouts to look elegant across various screen sizes.

---

## 🛠️ Tech Stack
*   **Structure:** HTML5
*   **Styling:** CSS3 (Custom animations, layouts)
*   **Logic & Animation Libraries:** JavaScript, jQuery, and specialized asynchronous script components (`love.js`, `function.js`)
